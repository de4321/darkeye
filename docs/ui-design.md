# 界面功能设计


每个界面就是ui.page下的xxxxPage.py

## 主页面
封面是一个AV推荐系统，包含每日推荐，故事，封面
- 推荐的影片，根据推荐算法推荐5部供选择，然后可以点直接观看
- 搜索番号后跳转作品详细界面+多搜索结果的显示
- 搜索女优名字后跳转女优详细界面+多搜索结果的显示
- 搜索男优，毕竟有的男优实在是太丑了
- 搜索女优的服装然后弹影片多搜索结果的显示，比如比基尼，服装颜色
- 搜索剧情然后弹影片多搜索结果的显示
- 搜索体位与联合卡片搜索
- 搜索做爱场景
- 搜索题材，比如护士，女OL，婚纱等等
- 开始观看后弹出本地的片源
- [ ] 封面竖排的功能

主页推荐设置模式，全随机，未看优先随机，智能推荐，养生模式
